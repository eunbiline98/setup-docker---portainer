# Docker Setup
Docker adalah platform kontainerisasi yang memungkinkan aplikasi berjalan di dalam kontainer terisolasi yang mencakup kode, dependensi, dan konfigurasi. Ini memastikan aplikasi tetap konsisten di berbagai lingkungan dari pengembangan hingga produksi tanpa risiko perubahan perilaku atau kompatibilitas
